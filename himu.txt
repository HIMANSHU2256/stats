ALGORITHM

1. define a function linear search having two argument named as (arr,x) where arr s array and x is element to search
2. define array and take input x from user 
3. using for loop we iterate i from index 0 to len(arr)
4. if arr[i] equal to value of x it return index of x else 0
5. now print "element at index" value



CODE 


def linearsearch(arr,x):
   arr=[1,2,3,4,5]
   x=input("Value of X:")
   n=len(arr)
   for i in range(0,n)
      if arr[i]==x:
         return i
   return 0
print("Element found at index:" +str(linearsearch(arr,x)))


output:


Value of X: 4
Element found at index: 3



here the time complexity is O(n)
